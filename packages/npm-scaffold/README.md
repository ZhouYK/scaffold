# rt-imgtocanvas

## ImgToCanvas

属性列表

| 属性名 | 类型 | 默认值 | 作用说明
| :----- | :----- | :----- | :----- |
| style | object | {width: 200, height: 200} | 设置canvas的行内样式
| className | string | '' | 给canvas添加自定义类名
| width | number | 300 | 设置canvas元素的宽度（和样式宽度不一样）
| height | number | 300 | 设置canvas元素的高度 (和样式宽度不一样)
| file | 图片的File或者Blob对象 | null | 图片的文件对象
